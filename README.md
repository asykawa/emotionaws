# Emotion Detection FastAPI Project

Проект на **FastAPI** для определения эмоций текста. Поддерживает:

* Перевод текста с русского на английский
* Токенизацию и преобразование текста в индексы
* Предсказание эмоции с помощью обученной LSTM модели PyTorch
* Возврат предсказанной эмоции через API


## 🛠 Возможности проекта

* Обработка русского текста через перевод
* Предсказание эмоций текста с помощью обученной LSTM модели
* API для интеграции с фронтендом или чат-ботом
* Обработка пустого текста и ошибок перевода

## 📌 Примечания

* Модель можно дообучать на новых данных для улучшения качества
* Проект можно расширять добавлением новых эмоций или предобработки текста
* Подходит для интеграции в приложения анализа текста, чат-боты и сервисы рекомендаций
